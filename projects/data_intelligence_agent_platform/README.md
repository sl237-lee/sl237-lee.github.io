# Data Intelligence Agent Platform

An AI-powered analytics copilot that combines:
- agent orchestration
- SQL generation
- vector retrieval
- analytics explanation
- dashboards

## Project Structure
- `api/` FastAPI backend
- `agents/` agent orchestration and task-specific agents
- `services/` SQL, retrieval, charting services
- `db/` local database schema and seed data
- `vectorstore/` embeddings and retrieval
- `ui/` Streamlit dashboard
- `docs/` architecture and roadmap
