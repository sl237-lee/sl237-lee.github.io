# AI Age Detector

Upload a photo to estimate age, gender, and emotion using deep learning.

## Quick Start
```bash
# Setup
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Run
python app.py
```

## Features

- Age estimation
- Gender prediction  
- Emotion detection
- Web interface with drag-and-drop
- Instant results

## Built with

- DeepFace (pre-trained models)
- Gradio (web interface)
- OpenCV (image processing)
